●【1_TakeTheNumberOrTakeTheSpeed】

　　・3年の進級制作展にて展示した3Dアクションゲームです。
　　・3Dの基礎学習を兼ねて制作しました。
　　・学習しながらも、できる限りの面白さの追求もしました。
　　・攻撃モーションについてもMMDで自作しました。

【制作期間】4ヶ月
【制作人数】1人
【動作環境】Window10
【開発環境】VisualStudio2019
【開発言語】C++
【使用ライブライ】DXライブラリ、ImGui
【フォルダ構成】・exeフォルダ
　　　　　　　　　実行データが入っているフォルダです。
　　　　　　　　　TakeTheNumberOrTakeTheSpeed.exeを
　　　　　　　　　ダブルクリックで起動できます。
　　　　　　　　 ---------------------------------------------　　
　　　　　　　　・projectフォルダ
　　　　　　　　　作品のプロジェクトフォルダです。
　　　　　　　　　TakeTheNumberOrTakeTheSpeed.slnを
　　　　　　　　　ダブルクリックでプロジェクトを起動できます。
                 ---------------------------------------------
　　　　　　　　・ProjectProposalフォルダ
　　　　　　　　　作品の企画書が入っています。
                 ---------------------------------------------
　　　　　　　　・@open.bat
                  ダブルクリックすると、最新のVisualStudioで
　　　　　　　　　プロジェクトを開くことができます。
　　　　　　　　 --------------------------------------------- 
                ・Game.mp4
                  作品のプレイ動画です。
　　　　　　　　 ---------------------------------------------
                ・ImGui.mp4
                  ImGuiの操作動画です。
　　　　　　　　 ---------------------------------------------
